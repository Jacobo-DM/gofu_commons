# Gofu Commons

Gofu is a bean-based, plant-protein food that can be made almost anywhere with simple tools — inspired by tofu, but freed from reliance on a single crop.

This repository holds the **Gofu Commons** website: a small family of recipes, teaching materials, stories, and a commons declaration, all released into the public domain under **CC0 1.0 Universal**.

## Origin

Gofu began in a family kitchen. The question: what would happen if tofu-making technology were opened up beyond soybeans? That question led to small experiments, and the hope that something nourishing, affordable, and more ecologically flexible might emerge — not as a product, but as a food commons.
