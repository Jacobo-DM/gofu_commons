# Gofu Commons

Gofu is a bean-based, plant-protein food that can be made almost anywhere with simple tools.

This repository holds the **Gofu Commons Declaration** and a small family of recipes and teaching files, all released under **CC0 1.0 Universal (public domain)**.

## Contents

- `index.html` – a simple landing page for the Gofu Commons
- `LICENSE` – CC0 1.0 Universal notice
- `docs/commons-declaration.md` – full text of the Gofu Commons Declaration
- `docs/story-luna-miami.md` – story-thread of how Gofu enters the SNMK / Discovery Mode universe
- `recipes/gofu-base.md` – base Gofu recipe
- `recipes/gofu-kudzu.md` – kudzu-enhanced Gofu
- `recipes/gofu-families.md` – ideas and notes for home cooking
- `recipes/gofu-schools.md` – notes for school kitchens and Discovery Mode modules
